# Assignment 01 — Basic Data Exploration and Cleaning using Pandas

## Overview

This project demonstrates the complete workflow of data exploration and preprocessing using Python and Pandas on an ecommerce dataset. The assignment focuses on understanding raw data, improving data quality, and preparing the dataset for downstream analytical tasks.

## Objective

To perform exploratory data analysis (EDA), handle missing values, clean inconsistencies, engineer a new feature, and export the processed dataset.

---

## Dataset Information

Dataset: Ecommerce Dataset (Products & Sizes Included)

File Used:
Combined_dataset.csv

Domain:
Ecommerce / Retail Analytics

---

## Project Workflow

### 1. Data Import
* Loaded CSV dataset into Pandas DataFrame

### 2. Data Exploration
* Previewed dataset records
* Checked dimensions
* Examined columns and datatypes
* Evaluated dataset structure

### 3. Data Cleaning
* Identified null values
* Processed missing values
* Removed duplicate records

### 4. Data Transformation
* Selected useful attributes
* Applied filtering operations
* Created derived feature:
  total_amount

### 5. Export
* Generated cleaned dataset:
  cleaned_dataset.csv
